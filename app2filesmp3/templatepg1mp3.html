How the Converter Works

This app uses the Python library pydub and the ffmpeg program to decode MP3 files and re-encode them as WAV files.

  1. You upload an MP3.
  2. The server saves it temporarily.
  3. pydub reads the MP3 and exports a WAV version.
  4. The browser automatically downloads the new WAV file.
